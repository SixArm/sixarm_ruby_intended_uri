Get and set the user’s intended uri in the Rails session array.

Example

  require 'intended_uri'
  class ApplicationController
    ...
    include IntendedUri
    ...
    def sign_in_success
      redirect_to intended_uri || welcome_uri
    end
    ...
    def sign_in_failure
      intended_uri=request.request_uri
      redirect_to sign_in
    end
